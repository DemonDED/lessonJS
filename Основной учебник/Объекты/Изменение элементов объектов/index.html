Изменение элементов объектов

Элементы объектов можно изменять, обращаясь к ним по ключам. Давайте посмотрим, как это делается:

let obj = {a: 1, b: 2, c: 3};
obj['a'] = '!';
или
obj.b = '!';
console.log(obj); // выведет {a: '!', b: 2, c: 3}

Задача 1: Дан объект: let obj = {x: 1, y: 2, z: 3}; Возведите в квадрат каждый элемент этого объекта.

Добавление элементов в объект

В объекты можно добавлять новые элементы, записывая их в нужные ключи. Давайте посмотрим, как это делается.

let obj = {};
obj['a'] = 1;
obj['b'] = 2;
obj['c'] = 3;
или
obj.a = 1;
obj.b = 2;
obj.c = 3;
console.log(obj); // выведет {a: 1, b: 2, c: 3}

Задача 2: Создайте пустой объект, а затем заполните его значениями.
