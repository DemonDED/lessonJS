Возведение в степень

Для возведения числа в степень также существует специальный оператор **.

alert(10 % 3); // выведет 1
let a = 10;
alert(a ** 3); // выведет 1000

Задача 1: Возведите число 2 в 10-ю степень. Выведите результат на экран.
