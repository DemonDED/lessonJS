Строки в JS

Строки создаются с помощью кавычек:

let str = 'abc';
alert(str); // выведет 'abc'

Кавычки могут быть не только одинарными, но и двойными.
Между одинарными и двойными кавычками в JavaScript нет никакой разницы.
