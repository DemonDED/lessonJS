Многомерные объекты

Многомерными могут быть не только массивы, но и объекты, вот пример:

let obj = {
a: {
key1: 'a1',
key2: 'a2',
key3: 'a3',
},
b: {
key1: 'b1',
key2: 'b2',
key3: 'b3',
},
c: {
key1: 'c1',
key2: 'c2',
key3: 'c3',
},
}

Давайте выведем какие-нибудь элементы нашего объекта:

console.log(obj['a']['key1']); // выведет 'a1'
console.log(obj.a.key1); // выведет 'a1'
console.log(obj['a'].key1); // выведет 'a1'

for (let key in obj) {
let subObj = obj[key];

for (let subKey in subObj) {
console.log(subObj[subKey]);
}
