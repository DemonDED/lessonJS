Многомерные структуры

Можно комбинировать многомерные массивы и объекты, создавая различные структуры. Давайте, к примеру, сделаем объект, содержащий внутри себя массивы:

let days = {
'ru': ['пн', 'вт', 'ср', 'чт', 'пт', 'сб', 'вс'],
'en': ['mn', 'ts', 'wd', 'th', 'fr', 'st', 'sn'],
};

Выведем какие-нибудь элементы из этого объекта:

console.log(days['ru'][0]); // выведет 'пн'
console.log(days['en'][2]); // выведет 'wd'

Задача 1: Дан следующий объект со студентами:

let students = {
'group1': ['name11', 'name12', 'name13'],
'group2': ['name21', 'name22', 'name23'],
'group3': ['name31', 'name32', 'name33'],
};

Выведите на экран первого студента из третьей группы.

Перебор многомерных структур

Пусть у нас дан следующий объект со студентами:

let students = {
'group1': ['student11', 'student12', 'student13'],
'group2': ['student21', 'student22', 'student23'],
'group3': ['student31', 'student32'],
};

Давайте выведем на экран имена всех студентов, перебрав наш объект двумя вложенными циклами.

Как вы видите, у нас объект с массивами. Это значит, что первый цикл должен быть по объекту, а второй - по массивам. То есть первый цикл будет for-in, а второй - for-of, вот так:

for (let group in students) {
for (let name of students[group]) {
console.log(name);
}
}

Ключи из переменных в многомерных структурах

Пусть теперь язык и номер дня хранятся в переменных:

let obj = {
'ru': ['пн', 'вт', 'ср', 'чт', 'пт', 'сб', 'вс'],
'en': ['mn', 'ts', 'wd', 'th', 'fr', 'st', 'sn'],
};
let lang = 'ru';
let day = 3;
console.log(obj[lang][day]);

Добавление элементов в многомерные массивы

Для этого параметром метода push передадим новый объект с юзером:

let users = [
{
name: 'name1',
surn: 'surn1',
},
{
name: 'name2',
surn: 'surn2',
},
{
name: 'name3',
surn: 'surn3',
},
];

users.push({
name: 'name4',
surn: 'surn4',
});
