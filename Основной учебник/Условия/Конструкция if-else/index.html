Конструкция if-else

Сейчас мы изучим специальную конструкцию if-else, которая позволяет выполнять некоторый код в зависимости от выполнения какого-либо условия.

С помощью конструкции if-else мы можем задать программе, к примеру, вопрос "больше ли переменная a, чем переменная b", и в зависимости от ответа выполнить или один код, или другой.

Операторы больше и меньше

Для проверки того, которое из значений больше, а которое меньше, используются операторы больше >, больше или равно >=, меньше <, меньше или равно <=.

Задача 2: Проверьте, что переменная test меньше 10.

Задача 3: Проверьте, что переменная test больше или равна 10.

Задача 4: Проверьте, что переменная test меньше или равна 10.

Проверка на равенство

Для проверки двух значений на равенство используется оператор ==.

Задача 5: Проверьте, что переменная test равна 10.

Проверка на неравенство

Оператор != проверяет на неравенство.

Задача 6: Проверьте, что переменная test не равна 10.

Сравнение переменных

В примерах выше мы сравнивали переменную с каким-то числом. Но никто не запрещает нам сравнивать две переменные.

let test1 = 1;
let test2 = 2;
if (test2 > test1) {
console.log('+++'); // сработает это
} else {
console.log('---');
}

Задача 8: Даны переменные test1 и test2. Проверьте, равны ли их значения и выведите соответствующее сообщение.

Равенство строк

При использовании конструкции if можно сравнивать и строки. Для примера пусть в переменной test хранится какая-то строка, например 'abc':

let test = 'abc';
if (test == 'abc') {
console.log('+++'); // сработает это
} else {
console.log('---');
}

Задача 9: Даны переменные test1 и test2, содержащие строки. Проверьте, равны ли их значения и выведите соответствующее сообщение.

Равенство строк и чисел

Как вы знаете, число в кавычках представляет собой строку. Например, '3' - это строка. Однако, при сравнении таких строк с настоящими числами JavaScript считает, что строка в кавычках равна такому же числу.

Задача 10: Даны переменные: let test1 = '123'; let test2 = 123; Проверьте, равны ли значения переменных и выведите соответствующее сообщение.

Равенство по значению и типу

Пусть вы хотите сравнивать так, чтобы число в кавычках не было равно такому же числу без кавычек. Говоря другими словами, вы хотите сравнивать так, чтобы сравнение было не только по значению, но и по типу данных.

Для этого вместо оператора == следует использовать оператор ===.

if ('3' === 3) {
console.log('+++');
} else {
console.log('---'); // сработает это
}

А вот при сравнении двух строк '3' условие будет истинно:
if ('3' === '3') {
console.log('+++'); // сработает это
} else {
console.log('---');
}

Неравенство по значению и типу

Кроме оператора != существует также оператор !==, учитывающий тип при сравнении.

if ('3' !== 3) {
console.log('+++'); // сработает это
} else {
console.log('---');
}
