Сокращенная форма проверки на истину

При программировании такие проверки требуются очень часто, поэтому для них существует более изящная сокращенная форма: вместо if (test == true) можно написать просто if (test).

Задача 1: Перепишите следующий код с использованием сокращенной формы:
let test = true;

if (test === true) {
console.log('+++');
} else {
console.log('---');
}

Сокращенная форма проверки на ложь

Вместо if (test !== true) можно написать просто if (!test).

Задача 2: Перепишите следующий код с использованием сокращенной формы:
let test = true;

if (test == false) {
console.log('+++');
} else {
console.log('---');
}

Задача 4: Перепишите следующий код с использованием сокращенной формы:

Общая сокращенная форма

Сокращенная форма проверки работает и для других типов данных. В этом случае эти данные сначала преобразуются к булеву типу, а потом сравниваются с true.

if (test) эквивалентно в этом случае if (Boolean(test) === true).

Сложные условия в сокращенной форме

В сокращенной форме можно использовать операторы && и ||.

if (test1 && test2) или if (test1 || test2).

Задача 5: Перепишите следующий код в сокращенной форме:
let test1 = true;
let test2 = true;

if (test1 === true && test2 === true) {
console.log('+++');
} else {
console.log('---');
}
