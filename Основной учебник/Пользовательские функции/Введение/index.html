Основы работы с пользовательскими функциями

Функция создается с помощью команды function. Далее через пробел следует имя функции, круглые скобки, а затем фигурные скобки, в которых пишется какой-то код:

function func() {
// какой-то код
}

Давайте теперь вызовем нашу функцию. Для этого необходимо написать ее имя и круглые скобки:

function func() {
console.log('!');
}

// Вызываем нашу функцию:
func(); // выведет '!'

Можно вызвать нашу функцию несколько раз, функции могут быть вызваны до места своего определения:

func(); // выведет '!'

function func() {
console.log('!');
}

Задача 1: Сделайте функцию, выводящую в консоль ваше имя.

Задача 2: Сделайте функцию, выводящую в консоль сумму чисел от 1 до 100.

Параметры функций

Функции могут принимать параметрами данные. Для примера давайте сделаем функцию, которая параметром будет принимать число:

function func(num) {
console.log(num ** 2);
}

func(2); // выведет 4
func(3); // выведет 9

Задача 3: Сделайте функцию, которая параметром принимает число и выводит в консоль куб этого числа.

Задача 4: Сделайте функцию, которая параметром принимает число и проверяет, положительное это число или отрицательное. В первом случае пусть функция выводит в консоль текст '+++', а во втором '---'.

Несколько параметров функций

Функция может принимать не один параметр, а несколько. В этом случае их нужно перечислить через запятую.

function func(num1, num2) {
console.log(num1 + num2);
}

func(1, 2); // выведет 3

Задача 5: Сделайте функцию, которая параметрами принимает 3 числа и выводит в консоль сумму этих чисел.

Параметры-переменные функций

В параметры функции можно передавать переменные.

function func(num) {
console.log(num ** 2);
}

let param = 2;
func(param);

Задача 6: Сделайте функцию func, которая параметром будет принимать 3 числа и выводить в консоль их сумму. Пусть даны 3 переменные с числами:

let param1 = 1;
let param2 = 2;
let param3 = 3;

С помощью созданной вами функции выведите в консоль сумму значений эти переменных.

Необязательные параметры

Параметры функций можно делать не обязательными. Для этого параметрам нужно задать значения по умолчанию.

function func(num = 0) {
console.log(num ** 2);
}

func(2); // выведет 4
func(); // выведет 0
