Добавление элементов через push

С помощью специального метода push можно добавлять элементы в конец массива. Пример:

let arr = [];
arr.push('a');
arr.push('b');
arr.push('c');
console.log(arr); // выведет ['a', 'b', 'c']

Задача 1: Заполните массив числами 1, 2 и 3.

Задача 2: Пусть дан такой массив: let arr = [1, 2, 3]; Добавьте ему в конец элементы 4 и 5.
