Получение элементов массива

Обращение к элементам массива осуществляется подобно обращениям к символам строки. Каждый элемент массива имеет свой номер: первый элемент - номер 0, второй - номер 1 и так далее. Эти номера называются ключами элементов массива.

let arr = ['a', 'b', 'c'];
console.log(arr[0]); // выведет 'a'
console.log(arr[1]); // выведет 'b'
console.log(arr[2]); // выведет 'c'

Задача 1: Создайте массив с элементами 1, 2, 3. Выведите на экран каждый из этих элементов.

Задача 1: Дан следующий массив:
let arr = [1, 2, 3];
Выведите на экран каждый из его элементов.

Задача 1: Дан следующий массив:
let arr = [1, 2, 3];
Выведите на экран сумму элементов этого массива.

Задача 1: Дан следующий массив:
