Длина массива

Длина массива находится с помощью свойства length:

let arr = ['a', 'b', 'c'];
console.log(arr.length) // выведет 3

Зная длину массива, можно получить его последний элемент:

let arr = [1, 2, 3];
console.log(arr[arr.length - 1]) // выведет 'c'
